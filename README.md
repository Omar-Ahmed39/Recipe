## ğŸ“ Overview

**Recipe App** is a fast and responsive web application designed for food enthusiasts. Built using **React** and **Vite**, it allows users to explore diverse food categories (Beef, Chicken, Seafood, etc.), and view detailed cooking instructions. The app fetches real-time data from a recipe API to provide a dynamic experience.

---

## âœ¨ Key Features

* **Category Navigation:** Explore meals based on categories like Area, Ingredients, or Type.
* **Recipe Details:** Get full ingredients lists, measures, and step-by-step instructions.
* **Modern UI:** A clean, sidebar-based navigation for easy access.
* **Responsive Design:** Optimized for all devices using Tailwind CSS.
